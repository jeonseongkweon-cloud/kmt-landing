DOUBLE CROSS 연계 프로그램

DOUBLE CROSS는 “구조 >> 교차원리, 교차원리 기반 무도”를 바탕으로 양손 협응·균형·반응을 강화하는 확장 훈련입니다. 유치부/초등부의 기본기 기반 위에 ‘재미 + 집중 + 성장’을 자연스럽게 연결합니다.

프로그램 허브 유치부 초등부 카톡 문의 DOUBLECROSS KOREA(연결 예정)
왜 DOUBLE CROSS인가?

아이들의 “몸의 연결(협응)”을 키우면 기본기 습득이 빨라지고, 자신감이 안정적으로 올라갑니다.

어떻게 연결되나?

유치부: 리듬·균형·양손 협응 → 초등부: 반응·거리감·정확도 → 심화: 구조화된 교차 동작.

운영 방식

개별 아이의 수준에 맞춰 단계형으로 안내합니다. 무리하게 ‘어렵게’ 진행하지 않습니다.
